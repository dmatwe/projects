**Тема:**

Регистрация клиента и автомобиля, осмотр, составление плана ремонта формирование заказ наряда, приемка авто

**Описание:**

Проект направлен на создание и внедрение комплексной системы управления бизнес-процессами, которая позволит эффективно осуществлять весь жизненный цикл заказов-нарядов. 

**Цель проекта:**

Упростить и автоматизировать процессы, связанные с регистрацией клиентов, управлением и редактированием данных клиентов, созданием, отслеживанием и редактированием заказов-нарядов, расчетом стоимости работ и запчастей, а также прозрачным взаимодействием между участниками команды и контролем наличия запчастей на складе.

**Основные функциональности проекта:**

1. Регистрация клиента для оформления заказа-наряда.
2. Изменение данных клиента с возможностью обновления информации в заказе-наряде.
3. Исключение ошибок человеческого фактора при формировании заказа-наряда.
4. Редактирование заказа-наряда для исправления ошибок.
5. Просмотр актуального статуса заказа-наряда для планирования дальнейших работ.
6. Получение предварительного расчета по стоимости всех работ для согласования заказа-наряда с клиентом.
7. Добавление запчастей в заказ-наряд для согласования фактуры с клиентом.
8. Замена отсутствующих запчастей на аналоги в заказе-наряде для ремонта автомобиля.
9. Получение предварительного расчета по стоимости всех запчастей для согласования заказа-наряда с клиентом.
10. Просмотр всех участников, работавших с заказом-нарядом, для оперативного взаимодействия в команде.
11. Просмотр наличия запчастей на локальном складе для формирования заявок на недостающие запчасти для заказа-наряда.
12. Заказ недостающих запчастей с центрального склада для обеспечения наличия необходимых запчастей на локальном складе.
13. Передача заказа-наряда в другую автомастерскую для выполнения всех необходимых работ по автомобилю.

**Ожидаемые результаты:**

1. Оптимизация бизнес-процессов автомастерской за счет автоматизации и упрощения действий.
2. Сокращение ошибок и повышение точности информации в заказах-нарядах.
3. Улучшение планирования работ и контроля статуса заказа-наряда.
4. Увеличение прозрачности и оперативности взаимодействия в команде.
5. Улучшение контроля над наличием запчастей на складе и снижение времени ожидания возможных доставок.


**Задачи из постановки могут повлиять на следующие метрики бизнес процессов автомастерских/СТО:**

1. Уровень удовлетворенности клиентов
2. Количество повторных обращений клиентов
3. Уровень прибыли
4. Трудозатраты на документооборот
